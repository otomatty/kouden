# バージョン 1.2.0 リリースノート

このバージョンでは、プロジェクトの透明性と情報共有を向上させる新機能を追加しました。

## 🎉 新機能

### マイルストーン機能
プロジェクトの将来的な開発計画を視覚的に確認できるようになりました。

<Accordion title="マイルストーン機能の詳細" defaultOpen={true}>

- **横スクロールタイムライン**: 時系列で計画を確認
- **進捗表示**: リアルタイムで進捗状況を確認
- **ステータス管理**: 計画中、進行中、完了のステータス
- **優先度表示**: 高・中・低の優先度設定

</Accordion>

### 更新履歴機能
過去のリリース履歴を縦スクロールタイムラインで確認できます。

<Accordion title="更新履歴機能の詳細">

- **セマンティックバージョニング**: major.minor.patch形式
- **カテゴリ分類**: 新機能、バグ修正、セキュリティ、パフォーマンス
- **破壊的変更表示**: 重要な変更を明確に表示
- **年別グループ化**: リリース年でグループ分け

</Accordion>

### アコーディオンコンポーネント
MDXファイル内で折りたたみ可能なセクションを作成できます。

```mdx
<Accordion title="詳細情報">
  ここに折りたたみたいコンテンツを記述
</Accordion>
```
