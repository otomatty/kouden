# 香典・供物の管理

## 記録できる情報

### 香典の場合
- 金額
- 贈り主の情報（氏名、続柄、住所など）
- 受け取り日時
- お礼状の送付状況
- メモ（任意）

### 供物の場合
- 品名
- 数量
- 贈り主の情報（氏名、続柄、住所など）
- 受け取り日時
- お礼状の送付状況
- メモ（任意）

## 一覧表示とフィルタリング

### 表示オプション
- リスト表示：簡潔な一覧で確認
- カード表示：詳細情報をカードで表示
- 表形式：スプレッドシートのような表で管理

### フィルタリング機能
- 種類（香典/供物）での絞り込み
- 金額範囲での検索
- 贈り主名での検索
- お礼状送付状況での絞り込み

## データの集計

### 自動集計機能
- 香典の合計金額
- 供物の種類別集計
- 日付別の集計
- お礼状の送付状況の集計

### グラフ表示
- 日別の記録数推移
- 金額の分布
- 種類別の割合

## お礼状管理

### お礼状の作成支援
- テンプレートからの作成
- 宛名印刷機能
- 送付状況の管理

### 一括処理
- 複数のお礼状をまとめて作成
- 送付状況の一括更新
- 印刷リストの出力
