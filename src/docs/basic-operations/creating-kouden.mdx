# 香典帳の作成方法

## 新しい香典帳を作成する

### 基本的な手順

1. **香典帳一覧ページにアクセス**
   - ログイン後、自動的に香典帳一覧ページが表示されます
   - 既存の香典帳がある場合は、それらが一覧で表示されます

2. **「香典帳を作成」ボタンをクリック**
   - ページ上部または中央にある青い「香典帳を作成」ボタンをクリックします
   - 作成フォームが表示されます

3. **必要な情報を入力**
   - **香典帳名**：故人の名前や式の名称を入力します（例：「山田太郎様 香典帳」）
   - **説明**（任意）：補足情報があれば入力します

4. **作成ボタンをクリック**
   - 入力内容を確認して「作成」ボタンをクリックします
   - 作成が完了すると、新しい香典帳のページに移動します

## 香典帳の設定

### 基本設定

新しく作成された香典帳では、以下の設定が自動的に行われます：

- **所有者権限**：作成者が自動的に所有者として設定されます
- **メンバー招待**：他のユーザーを招待してメンバーとして追加できます
- **デフォルト関係性**：基本的な関係性（親族、友人、会社関係など）が自動で設定されます

### プラン設定

香典帳には以下のプランがあります：

- **無料プラン**：基本的な機能をご利用いただけます
- **有料プラン**：より高度な機能や大容量での利用が可能です

### 初期設定のポイント

1. **香典帳名は後から変更可能**
   - 作成後でも設定画面から変更できます

2. **関係性の追加・編集**
   - デフォルトの関係性に加えて、独自の関係性を追加できます
   - 設定 > 関係性管理から編集可能です

3. **返礼品マスターの設定**
   - 返礼品を管理する場合は、あらかじめ返礼品マスターを設定しておくと便利です

## よくある質問

### Q: 香典帳は何個まで作成できますか？
A: 無料プランでは制限があります。詳細は料金プランをご確認ください。

### Q: 作成した香典帳を削除できますか？
A: はい。設定画面から削除（アーカイブ）することができます。

### Q: 他の人と香典帳を共有できますか？
A: はい。メンバー招待機能を使用して、他のユーザーと共有できます。
