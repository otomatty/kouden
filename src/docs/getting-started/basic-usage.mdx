# 基本的な使い方

このページでは、香典帳アプリの **基本的な操作方法** を、
初めてご利用いただく方にも分かりやすく説明します。

## 📱 アプリの画面構成

### メイン画面の見方

香典帳アプリにログインすると、以下の要素が画面に表示されます：

**上部ナビゲーション**
- 🏠 **ホーム**：メインダッシュボード
- 📚 **香典帳一覧**：作成済みの香典帳を表示
- ➕ **新規作成**：新しい香典帳を作成
- 👤 **アカウント**：設定とプロフィール

**サイドメニュー**（デスクトップ版）
- 各香典帳へのクイックアクセス
- 最近の活動履歴
- ショートカット機能

## 🆕 香典帳の作成

### ステップ1：新規作成の開始

1. **「新規作成」ボタンをクリック**
   - 画面上部の青い「新規作成」ボタンを探してクリック
   - またはダッシュボードの「＋新規作成」カードをクリック

2. **香典帳作成フォームが表示されます**

### ステップ2：基本情報の入力

**必須項目**
- **香典帳名**：「〇〇家告別式」「△△さん法要」など、分かりやすい名前
- **故人名**：香典帳の対象となる故人のお名前
- **開始日**：お通夜・告別式・法要の日付

**任意項目**
- **終了日**：法要期間がある場合
- **会場名**：セレモニーホール名など
- **メモ**：特記事項があれば

### ステップ3：アクセス設定

**プライバシー設定**
- **個人利用**：自分だけがアクセス可能
- **家族共有**：家族メンバーを招待して共同管理
- **団体利用**：複数の関係者で管理

### ステップ4：作成完了

「作成」ボタンをクリックすると、新しい香典帳が作成され、
詳細ページに自動的に移動します。

## 📝 香典・供物の記録

### 香典の記録方法

#### ステップ1：記録追加の開始

1. 香典帳の詳細ページで **「記録を追加」** ボタンをクリック
2. **「香典」** タブが選択されていることを確認

#### ステップ2：基本情報の入力

**必須情報**
- **金額**：受け取った香典の金額（数字のみ入力）
- **お名前**：香典をくださった方のフルネーム
- **ご関係**：故人との関係（親族、友人、会社関係など）

**詳細情報**（推奨）
- **ふりがな**：お名前の読み方
- **ご住所**：お礼状送付用の住所
- **電話番号**：連絡先（任意）
- **会社名・部署**：該当する場合

#### ステップ3：参列情報の記録

- **参列方法**：
  - 🏛️ 葬儀参列
  - 🚪 弔問のみ
  - 📮 郵送・代理
  - ❌ 欠席（香典のみ）

- **参列日時**：実際に参列された日時

#### ステップ4：その他の情報

**メモ欄（任意）**
- 特別な配慮事項
- 故人との思い出
- 今後の連絡事項

**写真添付（任意）**
- 香典袋の写真
- のし袋の表書き写真

### 供物の記録方法

#### 供物の種類

**お花**
- 供花：葬儀用の花輪・花束
- 枕花：故人の枕元に供える花

**弔電・電報**
- 弔電の内容と送り主

**その他供物**
- 果物、お菓子、線香等

#### 記録手順

1. **「記録を追加」** → **「供物」** タブを選択
2. **供物の種類** を選択
3. **送り主情報** を入力（香典と同様）
4. **詳細情報** を記録：
   - 品物の詳細（サイズ、種類など）
   - 配送業者・時間
   - 設置場所
5. **写真撮影**：供物の写真を記録

## 👥 参列者の管理

### 参列者記録の重要性

香典をいただかなくても、参列いただいた方の記録は重要です：
- お礼状の送付先確認
- 今後の関係維持
- 法要のご案内送付

### 参列者記録の手順

1. **「参列者」** タブを選択
2. **「参列者を追加」** ボタンをクリック
3. **基本情報** を入力：
   - お名前・ふりがな
   - ご関係
   - 参列日時
   - 連絡先

## 🔍 データの検索・確認

### 記録の検索方法

**基本検索**
- 画面上部の検索ボックスに名前を入力
- 部分一致で候補が表示されます

**詳細検索**
- 「詳細検索」をクリック
- 複数条件での絞り込み：
  - 金額範囲
  - 関係性
  - 参列方法
  - 記録日

### フィルタ機能

**金額順ソート**
- 金額の大きい順・小さい順で並び替え

**関係性別表示**
- 親族、友人、職場関係など、関係性別にグループ表示

**未対応項目の確認**
- お礼状未送付
- 住所不明
- 要確認項目

## ✏️ データの編集・削除

### 記録の編集方法

1. **編集したい記録を見つける**
2. **記録の右側の「編集」ボタンをクリック**
3. **内容を修正**
4. **「保存」ボタンをクリック**

### 削除の注意点

⚠️ **削除前の確認事項**
- 削除した記録は **復元できません**
- 重要なデータは事前にエクスポートを推奨
- 迷った場合は「メモ」欄に「要確認」と記載

### 削除手順

1. **削除したい記録の「削除」ボタンをクリック**
2. **確認ダイアログで「削除する」をクリック**
3. **記録が完全に削除されます**

## 📊 統計・集計の確認

### 自動集計される情報

**基本統計**
- 総受取額
- 記録件数
- 平均金額

**詳細分析**
- 関係性別集計
- 金額帯別分析
- 参列方法別統計

### 統計画面の見方

1. **「統計」タブをクリック**
2. **グラフとテーブルで情報を確認**
3. **期間指定**で分析範囲を調整可能

## 📤 データのエクスポート

### エクスポートの種類

**PDF出力**
- 印刷可能な美しいレポート
- お礼状作成用の住所録
- 会計報告書

**Excel/CSV出力**
- 表計算ソフトでの加工用
- 他システムへのデータ移行
- バックアップ用

### エクスポート手順

1. **「エクスポート」ボタンをクリック**
2. **出力形式を選択**（PDF/Excel/CSV）
3. **出力項目を指定**：
   - 全ての記録
   - 香典のみ
   - 供物のみ
   - 特定期間のみ
4. **「ダウンロード」ボタンをクリック**
5. **ファイルが自動ダウンロードされます**

## 👨‍👩‍👧‍👦 チーム機能（家族・関係者との共有）

### メンバー招待の手順

1. **「設定」** → **「メンバー管理」**を選択
2. **「メンバーを招待」**ボタンをクリック
3. **招待したい方のメールアドレスを入力**
4. **権限レベルを選択**：
   - **管理者**：すべての操作が可能
   - **編集者**：記録の追加・編集が可能
   - **閲覧者**：記録の確認のみ可能
5. **「招待送信」**ボタンをクリック

### チーム作業のメリット

- **作業分担**：複数人で同時に入力作業
- **リアルタイム同期**：変更がすぐに全員に反映
- **履歴管理**：誰がいつ何を変更したかを記録

## ❓ よくある質問と解決方法

### Q: 間違って記録を削除してしまいました
A: 申し訳ございませんが、削除された記録の復元はできません。重要なデータは定期的にエクスポートでバックアップを取ることをお勧めします。

### Q: 同じ人から複数回香典をいただいた場合は？
A: それぞれ別の記録として登録してください。システムが自動的に同一人物として認識し、統計に反映されます。

### Q: 金額が分からない場合は？
A: 「0」と入力するか、メモ欄に「金額不明」と記載してください。後で確認できたときに修正可能です。

### Q: 住所が分からない場合は？
A: 住所欄は空白でも構いません。分かる範囲の情報（都道府県のみなど）を入力し、メモ欄に「住所要確認」と記載することをお勧めします。

## 📞 サポートが必要な場合

### お困りの際は

- **ヘルプセンター**：よくある質問とマニュアル
- **チャットサポート**：リアルタイムでの質問対応
- **メールサポート**：詳細な問い合わせ
- **電話サポート**：緊急時の直接対応

### サポート営業時間

- **平日**：9:00 〜 18:00
- **土日祝日**：10:00 〜 17:00
- **緊急時**：24時間対応（技術的トラブルのみ）
