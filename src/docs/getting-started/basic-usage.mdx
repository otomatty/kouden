# 基本的な使い方

このページでは、香典帳アプリの **基本的な操作方法** を、
初めてご利用いただく方にも分かりやすく説明します。

## 📱 アプリの画面構成

### メイン画面の見方

香典帳アプリにログインすると、以下の要素が画面に表示されます：

**上部ナビゲーション**
- 🏠 **ホーム**：メインダッシュボード
- 📚 **香典帳一覧**：作成済みの香典帳を表示
- ➕ **新規作成**：新しい香典帳を作成
- 👤 **アカウント**：設定とプロフィール

**サイドメニュー**（デスクトップ版）
- 各香典帳へのクイックアクセス
- 最近の活動履歴
- ショートカット機能

## 🆕 香典帳の作成

### ステップ1：新規作成の開始

1. **「新規作成」ボタンをクリック**
   - 画面上部の青い「新規作成」ボタンを探してクリック
   - またはダッシュボードの「＋新規作成」カードをクリック

2. **香典帳作成フォームが表示されます**

### ステップ2：基本情報の入力

**必須項目**
- **香典帳名**：「〇〇家告別式」「△△さん法要」など、分かりやすい名前
- **故人名**：香典帳の対象となる故人のお名前
- **開始日**：お通夜・告別式・法要の日付

**任意項目**
- **終了日**：法要期間がある場合
- **会場名**：セレモニーホール名など
- **メモ**：特記事項があれば

### ステップ3：アクセス設定

**プライバシー設定**
- **個人利用**：自分だけがアクセス可能
- **家族共有**：家族メンバーを招待して共同管理
- **団体利用**：複数の関係者で管理

### ステップ4：作成完了

「作成」ボタンをクリックすると、新しい香典帳が作成され、
詳細ページに自動的に移動します。

## 📝 香典・供物の記録と管理

香典帳アプリの中心となる機能が、香典や供物の情報を記録し、管理することです。

### 香典の記録

香典を受け取ったら、その情報をアプリに記録します。
香典帳を開くと、登録されている香典の一覧が表示されます。

1.  **新規登録**:
    一覧画面の右上にある **「香典を登録する」** ボタンをクリックすると、新しい香典情報を入力するダイアログが開きます。
    詳しい入力項目や手順については、[香典の記録方法](../kouden-entries/recording-entries) のページで詳細に説明しています。

2.  **一覧の確認と操作**:
    登録済みの香典は一覧で確認できます。一覧では以下の操作が可能です。
    *   **表示件数の変更**: 一度に表示する件数を変更できます。
    *   **検索**: お名前や組織名などで記録を検索できます。
    *   **並び替え**: 金額や登録日などで一覧を並び替えることができます。
    *   **フィルター**: 関係性、参列状況、日付などで表示する記録を絞り込むことができます。
    *   **編集・削除**: 各記録の右側にあるメニューから、情報の編集や削除を行えます。

    これらの詳細な操作方法については、[香典一覧の操作方法](../kouden-entries/entry-management) （←新しいページへのリンク案）のページで解説します。

### 供物の記録

供花や弔電、その他お供え物などの情報も記録・管理できます。
香典と同様に、「供物」タブから記録の追加や一覧の確認が可能です。

#### 記録手順

1. 香典帳の詳細ページで、香典一覧の上部などにある **「供物」** タブを選択します。
2. **「供物を記録する」** （または同様のボタン）をクリックします。
3. **供物の種類**（例：供花、弔電、果物など）を選択します。
4. **送り主の情報**（お名前、連絡先など）を入力します。これは香典の記録時と同様です。
5. **供物の詳細情報**（品名、数量、いただいた日付、写真など）を記録します。
6. **「保存」** ボタンをクリックします。

詳細な供物の記録方法については、[供物の記録方法](../offerings/recording-offerings) のページを参照してください。

## 👥 参列者の管理 (現時点では香典・供物記録と統合)

現在のバージョンでは、独立した「参列者管理」機能は提供されておらず、香典や供物の記録を通じて参列された方の情報を管理します。
香典をいただいていない参列者の方の情報も、例えば金額を0円として香典記録に登録するなどして管理することが可能です。

## 🔍 データの検索・絞り込み

香典や供物の記録が増えてくると、特定の情報を探し出すのが大変になります。
本アプリでは、効率的にデータを見つけるための検索機能や絞り込み（フィルター）機能を提供しています。

### 香典一覧での検索・絞り込み

香典一覧画面では、画面上部やツールバー領域に検索ボックスやフィルターオプションが用意されています。

*   **キーワード検索**: お名前、会社名、住所の一部などを入力して、合致する記録を検索できます。
*   **日付フィルター**: 香典をいただいた日付の範囲を指定して絞り込むことができます。
*   **金額フィルター**: 香典の金額範囲を指定できます。
*   **関係性フィルター**: 故人との関係性（例：親族、友人）で絞り込めます。
*   **参列状況フィルター**: 参列タイプ（例：葬儀参列、弔問）で絞り込めます。
*   **重複チェックフィルター**: 重複の可能性がある記録のみを表示することができます。

これらの機能を組み合わせることで、必要な情報を素早く見つけ出すことができます。
詳細な検索・絞り込み機能の使い方は、[検索・フィルタ機能](../kouden-entries/search-and-filter) のページで詳しく解説しています。

## ✏️ データの編集・削除

### 記録の編集方法

1. **編集したい記録を見つける**
2. **記録の右側の「編集」ボタンをクリック**
3. **内容を修正**
4. **「保存」ボタンをクリック**

### 削除の注意点

⚠️ **削除前の確認事項**
- 削除した記録は **復元できません**
- 重要なデータは事前にエクスポートを推奨
- 迷った場合は「メモ」欄に「要確認」と記載

### 削除手順

1. **削除したい記録の「削除」ボタンをクリック**
2. **確認ダイアログで「削除する」をクリック**
3. **記録が完全に削除されます**

## 📊 統計・集計の確認

### 自動集計される情報

**基本統計**
- 総受取額
- 記録件数
- 平均金額

**詳細分析**
- 関係性別集計
- 金額帯別分析
- 参列方法別統計

### 統計画面の見方

1. **「統計」タブをクリック**
2. **グラフとテーブルで情報を確認**
3. **期間指定**で分析範囲を調整可能

## 📤 データのエクスポート

### エクスポートの種類

**PDF出力**
- 印刷可能な美しいレポート
- お礼状作成用の住所録
- 会計報告書

**Excel/CSV出力**
- 表計算ソフトでの加工用
- 他システムへのデータ移行
- バックアップ用

### エクスポート手順

1. **「エクスポート」ボタンをクリック**
2. **出力形式を選択**（PDF/Excel/CSV）
3. **出力項目を指定**：
   - 全ての記録
   - 香典のみ
   - 供物のみ
   - 特定期間のみ
4. **「ダウンロード」ボタンをクリック**
5. **ファイルが自動ダウンロードされます**

## 👨‍👩‍👧‍👦 チーム機能（家族・関係者との共有）

### メンバー招待の手順

1. **「設定」** → **「メンバー管理」**を選択
2. **「メンバーを招待」**ボタンをクリック
3. **招待したい方のメールアドレスを入力**
4. **権限レベルを選択**：
   - **管理者**：すべての操作が可能
   - **編集者**：記録の追加・編集が可能
   - **閲覧者**：記録の確認のみ可能
5. **「招待送信」**ボタンをクリック

### チーム作業のメリット

- **作業分担**：複数人で同時に入力作業
- **リアルタイム同期**：変更がすぐに全員に反映
- **履歴管理**：誰がいつ何を変更したかを記録

## ❓ よくある質問と解決方法

### Q: 間違って記録を削除してしまいました
A: 申し訳ございませんが、削除された記録の復元はできません。重要なデータは定期的にエクスポートでバックアップを取ることをお勧めします。

### Q: 同じ人から複数回香典をいただいた場合は？
A: それぞれ別の記録として登録してください。システムが自動的に同一人物として認識し、統計に反映されます。

### Q: 金額が分からない場合は？
A: 「0」と入力するか、メモ欄に「金額不明」と記載してください。後で確認できたときに修正可能です。

### Q: 住所が分からない場合は？
A: 住所欄は空白でも構いません。分かる範囲の情報（都道府県のみなど）を入力し、メモ欄に「住所要確認」と記載することをお勧めします。
