# 基本的な使い方

香典帳アプリでは、以下の基本的な操作を行うことができます。

## 香典帳の作成

1. ヘッダーの「新規作成」ボタンをクリック
2. 香典帳のタイトルを入力
3. 必要に応じて追加情報を入力
4. 「作成」ボタンをクリック

## 香典・供物の記録

1. 香典帳の詳細ページで「記録を追加」ボタンをクリック
2. 以下の情報を入力：
   - 記録の種類（香典・供物）
   - 金額または品名
   - 贈り主の情報
   - メモ（任意）
3. 「保存」ボタンをクリック

## 参列者の管理

1. 「参列者」タブを選択
2. 「参列者を追加」ボタンをクリック
3. 以下の情報を入力：
   - 氏名
   - 続柄
   - 参列日時
   - メモ（任意）
4. 「保存」ボタンをクリック

## データの編集と削除

- 各記録の右側にある「編集」ボタンから内容を変更できます
- 「削除」ボタンで記録を削除できます
- 削除した記録は元に戻せないので注意してください

## データのエクスポート

1. 香典帳の詳細ページで「エクスポート」ボタンをクリック
2. エクスポート形式（Excel/CSV）を選択
3. ダウンロードされたファイルを保存
