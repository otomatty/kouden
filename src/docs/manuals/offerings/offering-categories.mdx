# 供物の分類と管理

## 供物の種類（カテゴリ）

供物を記録する際は、あらかじめ用意された次の5つの種類から選択します。
種類を正しく選ぶことで、一覧での絞り込みや集計がしやすくなります。

| 種類 | 説明 |
| --- | --- |
| **供花** | 花輪・生花・盛花・花束などの花類 |
| **供物** | 果物・菓子などの品物 |
| **線香** | 線香・ろうそくなど |
| **御供物料** | 金銭による供物（御供物料） |
| **その他** | 上記に当てはまらない供物 |

> 種類はあらかじめ決められた5つの中から選ぶ形式です。
> 利用者が独自の種類（カスタムカテゴリ）を追加することは、現時点ではできません。

## 種類による絞り込み

供物の一覧（テーブル形式）では、種類で絞り込んで表示できます。
「供花だけを確認したい」といった場合に便利です。
また、種類・提供者名・内容での検索や並び替えにも対応しています。

## 種類の選び方のポイント

- **花類は「供花」**：花輪・生花・盛花・花束はまとめて「供花」を選びます
- **金銭は「御供物料」**：品物ではなく金銭でいただいた場合は「御供物料」を選びます
- **迷ったら「その他」**：分類に迷う場合は「その他」を選び、内容欄に詳細を記入してください

## よくある質問

### Q: 独自のカテゴリを追加できますか？
A: 現時点では種類の追加はできません。決められた5種類から選び、詳細は内容・備考欄に記入してください。

### Q: 弔電は供物として記録しますか？
A: 弔電は専用の[電報管理](../telegrams/recording-telegrams)で記録できます。供物として扱う場合は「その他」を選んでください。
