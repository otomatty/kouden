# 供物管理機能

供物管理は、いただいた供花・供物・御供物料などをまとめて管理する機能です。
ここでは、供物の入力項目や一覧表示、編集・削除、香典との関連付けなど、
管理全般について説明します。供物の基本的な記録手順は
[供物の記録方法](./recording-offerings)をご覧ください。

## 供物の入力項目

| 項目 | 必須 | 説明 |
| --- | --- | --- |
| **種類** | 必須 | 供花／供物／線香／御供物料／その他 |
| **提供者名** | 必須 | 供物の送り主（例：山田太郎） |
| **内容** | 任意 | 具体的な内容（例：胡蝶蘭） |
| **数量** | 必須 | 1〜999 |
| **金額** | 任意 | 概算金額（最大9,999,999円） |
| **備考** | 任意 | 補足メモ |
| **関連する香典情報** | 任意 | 関連する香典記録（複数選択可） |
| **写真** | 任意 | 供物の写真（複数枚） |

入力フォームは「基本情報」と「追加情報」のタブに分かれています。

## 一覧表示

供物の一覧は表示環境に合わせて切り替わります。

- **カード形式**：種類のバッジ・提供者名・内容・金額・数量が見やすく並びます
- **テーブル形式**：種類・提供者名・内容・金額・関連香典・数量・備考・写真などの列が並びます
  - 種類・提供者名・金額で並び替え、提供者名・内容・種類で検索、種類で絞り込みが可能です

## 編集・削除

- 一覧から供物を選ぶと詳細画面が開き、各項目を編集できます
- 「基本情報」「追加情報」のタブで項目が分かれています
- 削除は確認ダイアログ「お供物の削除」で確認のうえ実行します
  - 「このお供物を削除してもよろしいですか？ この操作は取り消すことができません。」
- テーブル形式では複数選択してまとめて削除することもできます

> ⚠️ 削除した供物は復元できません。

## 香典との関連付け

供物は、複数の香典記録に関連付けられます。
同じ方から香典と供物の両方をいただいた場合に紐づけておくと、
合計でのお礼・返礼の検討がしやすくなります。

## 配分管理

金額が登録されている供物は、「配分管理」から各香典記録へ金額を配分できます。
配分した金額は[統計機能の概要](../statistics/statistics-overview)の
「合計金額（配分込み）」や、返礼の合計金額の算出に反映されます。

## 操作できる権限について

供物の編集・削除は、香典帳の**オーナー**または**編集者**のみが行えます。
**閲覧者**は内容の確認のみ可能です。

## よくある質問

### Q: 1つの供物を複数の方の香典に紐づけられますか？
A: はい。供物には複数の香典記録を関連付けられます。

### Q: 配分管理は必ず行う必要がありますか？
A: 必須ではありません。供物の金額を香典と合算して返礼を検討したい場合に活用してください。
