# API連携機能

## 現在の対応状況

> ⚠️ 利用者がご自身で設定できる外部API連携機能は、現時点では提供していません。
> 今後の実装を予定している機能です。

香典帳アプリは内部的に決済（Stripe）やメール送信などの外部サービスと
連携して動作していますが、これらは利用者が直接設定・操作するものではありません。

現時点では、以下のような **利用者向けのAPI連携機能は実装されていません**。

- APIキーの発行・管理画面
- 外部サービスへのデータ連携（Webリクエスト）設定
- 公開API（外部プログラムからの香典帳データ操作）

## 今後実装予定の内容

将来的には、以下のような連携機能の提供を検討しています。

- 外部サービスとのデータ連携
- API経由でのデータの取得・登録

実装時期や仕様は決まり次第お知らせします。

## 現在ご利用いただける代替手段

データを他のツールで活用したい場合は、エクスポート機能をご利用ください。

- **CSV / Excel エクスポート**：表計算ソフトでの集計・加工に利用できます
- **PDF 出力**：印刷・共有用の一覧出力に利用できます

詳しくは[データのバックアップと復元](./backup-restore)および
[PDF出力機能](../statistics/pdf-export)をご覧ください。

## よくある質問

### Q: 香典帳のデータを外部システムに自動で送れますか？
A: 現時点では自動連携の機能はありません。CSV / Excel エクスポートでデータを取り出してご活用ください。
