# パフォーマンス問題の解決

香典帳アプリの動作が重い・読み込みが遅いと感じたときの、
原因の切り分けと対処方法を説明します。

## まず確認すること

### 通信環境を確認する
- 電波の弱い場所では読み込みが遅くなります。安定した通信環境でお試しください
- モバイル回線の場合はWi-Fiへの切り替えもお試しください

### ブラウザ・アプリを最新にする
- ご利用のブラウザを最新版に更新してください
- 別のブラウザでも同じ症状が出るか確認すると、原因の切り分けに役立ちます

### 再読み込み・再起動
- ページを再読み込みしてみてください
- ブラウザのタブを開き直す、端末を再起動するなども有効です

## 大量のデータを扱うとき

香典記録の一覧は、件数が多くても快適に表示できるよう、
スクロールに応じて少しずつ読み込まれる仕組み（無限スクロール）になっています。
返礼記録なども同様に、ページ単位で順次読み込まれます。

それでも動作が重いと感じる場合は、次の工夫が有効です。

- **検索・絞り込みで対象を絞る**：[検索・絞り込み機能](../kouden-entries/search-and-filter)で
  表示する件数を減らすと軽くなります
- **期間やメンバーで絞り込む**：必要な範囲だけを表示してください
- **不要なタブ・アプリを閉じる**：端末の負荷を下げてください

## 画像・写真について

供物の写真など画像を多く含むページは、読み込みに時間がかかることがあります。
- 通信環境の良い場所で表示してください
- 写真は1ファイルあたり最大5MBまでです。大きすぎる画像は避けてください

## それでも改善しない場合

症状が続く場合は、次の情報を控えてお問い合わせください。

- どの画面・操作で遅いと感じるか
- データ件数のおおよその規模
- ご利用の端末・ブラウザ・通信環境
