# エラーメッセージの対応

操作中にエラーが表示された場合の、代表的なメッセージと対処方法を説明します。
エラーは画面上部などに通知（トースト）やメッセージとして表示されます。

## よくあるエラーと対処方法

### 「認証が必要です」
ログインが必要な状態、またはログイン情報の有効期限が切れている可能性があります。
- 一度ログアウトして、再度ログインし直してください
- ログインページから操作をやり直してください

### 「この操作を行う権限がありません」
その操作を行う権限がありません。
- ご自身の役割（オーナー・編集者・閲覧者）を確認してください
- 閲覧者は記録の作成・編集・削除ができません。必要であれば香典帳のオーナーに権限の変更を依頼してください
- 詳しくは[役割と権限の管理](../permissions/role-management)をご覧ください

### 「対象のデータが見つかりませんでした」
対象のデータが既に削除されているか、アクセスできない状態の可能性があります。
- 画面を再読み込みして最新の状態を確認してください
- 別のメンバーが先に削除・変更していないか確認してください

### 「通信エラーが発生しました」
ネットワークの接続に問題がある可能性があります。
- インターネット接続を確認してください
- 時間をおいて再度お試しください

### 「通信がタイムアウトしました」
処理に時間がかかり、通信が時間切れになった可能性があります。
- 電波状況の良い場所で、時間をおいて再度お試しください

### 「予期せぬエラーが発生しました」
想定外のエラーが発生した状態です。
- 画面を再読み込みして操作をやり直してください
- 繰り返し発生する場合は、操作内容を控えてサポートまでお問い合わせください

## 入力内容に関するエラー

入力フォームでは、項目ごとに「○文字以内で入力してください」「必須です」などの
案内が表示されることがあります。表示された案内に従って入力内容を修正してください。

## エラーが解消しない場合

上記を試してもエラーが解消しない場合は、次の情報を控えてお問い合わせください。

- 表示されたエラーメッセージ
- どの操作をしたときに発生したか
- ご利用の端末・ブラウザ
