# レポート生成機能

## レポートとして使える機能

香典帳には、データを集計・出力して報告や共有に使える機能がそろっています。
「レポート」という独立した作成画面はありませんが、
以下を組み合わせることで報告書や控えとして活用できます。

### 統計サマリー
[統計機能の概要](./statistics-overview)では、合計金額・参列者数・返礼進捗などの
サマリーやグラフが自動で表示されます。
全体像を把握したいときの報告資料として利用できます。

### PDF出力
香典記録の一覧を、合計金額付きのPDFとして出力できます。
印刷や関係者への共有に適しています。
詳しくは[PDF出力機能](./pdf-export)をご覧ください。

### CSV / Excel エクスポート
香典記録の一覧を表計算ソフト向けのデータとして出力できます。
独自の集計や別資料への転記に活用できます（有料プラン）。

## 目的別のおすすめ出力

| 目的 | おすすめの方法 |
| --- | --- |
| 全体の傾向を把握したい | [統計機能の概要](./statistics-overview) |
| 印刷して配布・保管したい | [PDF出力機能](./pdf-export) |
| 表計算ソフトで集計したい | CSV / Excel エクスポート（有料プラン） |

## よくある質問

### Q: オリジナルのレイアウトでレポートを作成できますか？
A: 現時点ではレイアウトを自由に編集するレポート作成機能はありません。PDF・CSV・Excel の各出力をご活用ください。

### Q: 統計グラフをそのまま出力できますか？
A: 統計画面はアプリ内で確認できます。一覧データの出力は PDF / CSV / Excel をご利用ください。
