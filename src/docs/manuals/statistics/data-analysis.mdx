# データ分析の活用方法

香典帳に蓄積したデータは、統計画面や検索・絞り込み機能を使って
さまざまな角度から把握できます。

## 統計画面での分析

[統計機能の概要](./statistics-overview)の画面では、以下のような分析結果が
自動で表示されます。

### 金額の把握
- **合計金額（配分込み）**：香典といただいた供物の配分を合わせた合計
  - 香典分・お供物配分・合計の内訳が確認できます

### 参列状況の把握
- **参列者数**：葬儀・弔問・欠席の人数内訳
- **参列種別の円グラフ**：参列区分の割合を視覚的に確認

### 金額分布の把握
- **金額別分布の棒グラフ**：金額帯ごとの件数を確認
  - どの金額帯が多いかを把握でき、返礼品の検討に役立ちます

### 返礼の進捗
- **返礼進捗**：完了・一部返礼・未対応・返礼不要の割合と進捗率

## 検索・絞り込みでの分析

香典記録の一覧では、条件を指定して対象を抽出できます。
詳しくは[検索・絞り込み機能](../kouden-entries/search-and-filter)をご覧ください。

- **キーワード検索**：氏名・住所・団体名・役職での検索
- **並び替え**：登録の新しい順／古い順、金額が大きい順／小さい順
- **期間で絞り込み**：登録日の範囲指定
- **メンバーで絞り込み**：誰が登録したかで抽出
- **重複の確認**：重複の可能性がある記録の抽出

## 表計算ソフトでのさらなる分析

より詳細な集計やグラフ作成を行いたい場合は、
CSV / Excel エクスポート（有料プラン）でデータを取り出し、
表計算ソフトで自由に分析できます。
詳しくは[データのバックアップと復元](../advanced/backup-restore)をご覧ください。

## よくある質問

### Q: 関係性ごとの集計はできますか？
A: 一覧で関係性を確認できるほか、CSV / Excel に出力して関係性ごとに集計することもできます。

### Q: 金額帯ごとの件数を知りたい
A: 統計画面の「金額別分布」の棒グラフで確認できます。
