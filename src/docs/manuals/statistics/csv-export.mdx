# CSV出力機能

## CSV出力について

CSV出力機能を使用することで、香典帳の記録データを汎用的なCSV（Comma Separated Values）形式で出力できます。
軽量で多くのシステムと互換性があるため、データの連携や移行に最適です。
この機能は有料プランでご利用いただけます。

## 基本的なCSV出力

### 出力手順

1. **香典帳詳細ページ**
   - 香典帳詳細ページを開く
   - 出力対象の香典帳を確認

2. **ダウンロードメニューを開く**
   - 「ダウンロード」ボタンをクリック
   - 出力形式の選択画面が表示

3. **CSV形式を選択**
   - 「CSV(.csv)」オプションをクリック
   - 即座にダウンロードが開始

## CSV出力の特徴

### データ形式

1. **ヘッダー情報**
   - ご芳名、団体名、役職、ご関係
   - 金額、郵便番号、住所、電話番号
   - 参列、供物、備考

2. **文字エンコーディング**
   - UTF-8（BOM付き）
   - 日本語文字の文字化け防止
   - Excelでの開封時も正常表示

3. **データ形式の統一**
   - 金額は数値として出力
   - 参列タイプは日本語で表示
   - 供物の有無は「有」「無」で表示

### ファイル命名規則

```
{香典帳タイトル}_{出力日付}.csv
例：田中太郎様香典帳_2024-01-15.csv
```

## トラブルシューティング

### よくある問題と解決方法

#### Q: Excelで開くと文字化けする
**A: 以下の手順で解決できます**
1. Excelの「データ」タブから「テキストからファイルを取得」
2. ファイル形式で「区切り文字」を選択
3. エンコーディングを「UTF-8」に設定
4. 区切り文字で「カンマ」を選択

#### Q: ダウンロードが始まらない
**A: 以下をご確認ください**
- ブラウザのポップアップブロック設定
- ダウンロードフォルダの容量
- ネットワーク接続の状態
- 有料プランの契約状況

#### Q: データが途中で切れている
**A: 出力条件をご確認ください**
- フィルター条件の設定
- 期間指定の範囲
- 権限によるアクセス制限
