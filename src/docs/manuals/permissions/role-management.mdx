# 役割と権限の管理

## 役割の種類

香典帳は複数のメンバーで共有して作業できます。
各メンバーには次の3つの役割（権限）のいずれかが割り当てられます。

| 役割 | 説明 |
| --- | --- |
| **オーナー** | 香典帳を作成した本人。すべての操作が可能で、メンバーの管理や香典帳の設定変更も行えます |
| **編集者** | 香典・供物・電報・返礼などの記録を作成・編集・削除できます |
| **閲覧者** | 記録の閲覧のみ可能です。データの追加や変更はできません |

> オーナーは香典帳を作成した方に自動的に割り当てられる役割です。
> 招待したメンバーには「編集者」または「閲覧者」を割り当てます。

## 役割の割り当て

メンバーへの役割の割り当ては、香典帳の設定画面のメンバー管理から行います。
メンバーを招待する手順は[メンバーの招待と権限管理](../basic-operations/inviting-members)をご覧ください。

- 役割を変更できるのは**オーナー**のみです
- メンバーの追加・役割変更・削除はオーナーが管理します

## 役割ごとにできること（概要）

| 操作 | オーナー | 編集者 | 閲覧者 |
| --- | :---: | :---: | :---: |
| 記録の閲覧 | ✓ | ✓ | ✓ |
| 記録の作成・編集・削除 | ✓ | ✓ | − |
| メンバーの招待・役割変更・削除 | ✓ | − | − |
| 香典帳の設定変更 | ✓ | − | − |

詳しい権限の違いは[メンバー権限の詳細](./member-permissions)をご覧ください。

## よくある質問

### Q: オーナーを別の人に変更できますか？
A: オーナーは香典帳の作成者に紐づきます。運用方法については[メンバー権限の詳細](./member-permissions)もあわせてご確認ください。

### Q: 閲覧者を編集者に変更したい
A: オーナーがメンバー管理画面から役割を変更できます。
