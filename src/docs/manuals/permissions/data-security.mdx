# データセキュリティについて

香典帳には、ご芳名・住所・電話番号・金額といった個人情報や
大切な記録が含まれます。これらのデータを安全に扱うため、
本サービスでは以下の仕組みでデータを保護しています。

## アクセス制御

### 役割に基づくアクセス制限
香典帳のデータには、その香典帳のメンバーだけがアクセスできます。
さらに、メンバーの役割（オーナー・編集者・閲覧者）に応じて
できる操作が制限されます。
詳しくは[役割と権限の管理](./role-management)をご覧ください。

### 行レベルのアクセス制御
データベース側でも、ユーザーごと・香典帳ごとにアクセスできる範囲が
厳密に制御されています。
自分が関係しない香典帳のデータを他のユーザーが閲覧・編集することはできません。

## 通信の安全性

サービスとの通信はすべて暗号化（HTTPS）された経路で行われます。
入力したデータが通信途中で第三者に盗み見られないように保護されています。

## 認証

各操作はログインしているユーザーの認証情報に基づいて行われます。
認証されていない状態で保護されたデータにアクセスしようとすると、
ログインが求められます。

## 安全にご利用いただくために

データを安全に保つために、利用者側でも次の点にご注意ください。

- **パスワードを適切に管理する**：他人に推測されにくいパスワードを使用してください
- **共有メンバーを必要最小限にする**：本当に必要な方だけを招待し、不要になったメンバーは削除してください
- **役割を適切に設定する**：閲覧のみで十分な方には「閲覧者」を割り当ててください
- **共有端末でのログインに注意する**：共用のパソコンなどではログアウトを忘れないでください

## よくある質問

### Q: 入力したデータは他の利用者に見られますか？
A: いいえ。各香典帳のデータは、その香典帳のメンバーのみがアクセスできます。

### Q: アカウントを削除するとデータはどうなりますか？
A: アカウントや香典帳の削除に関する取り扱いは、サービスの利用規約・プライバシーポリシーをご確認ください。
