# アプリの基本的な操作方法

## 画面構成について

### メイン画面の構成

香典帳アプリは、分かりやすい構成で設計されています：

1. **ヘッダー**
   - アプリロゴとナビゲーション
   - ユーザーメニュー（プロフィール、設定、ログアウト）
   - 通知アイコン

2. **サイドバー**（デスクトップ版）
   - 香典帳一覧
   - 設定メニュー
   - ヘルプ・サポート

3. **メインコンテンツエリア**
   - 現在表示中のページのコンテンツ
   - 操作ボタンやフォーム

4. **フッター**
   - 利用規約、プライバシーポリシー
   - サポート情報

## 基本的なナビゲーション

### 香典帳間の移動

1. **香典帳一覧から選択**
   - ログイン後の香典帳一覧から、目的の香典帳をクリック
   - カード形式で表示される各香典帳を選択

2. **サイドバーからの切り替え**（デスクトップ版）
   - 左側のサイドバーに香典帳一覧が表示
   - クリックで即座に切り替え可能

3. **パンくずナビゲーション**
   - ページ上部のパンくずで現在地を確認
   - 上位階層への移動が簡単

### タブナビゲーション

香典帳詳細画面では、以下のタブで機能を切り替えます：

- **記録**：香典の記録・一覧
- **供物**：供物の記録・管理
- **電報**：電報の記録・管理
- **統計**：集計結果とグラフ
- **設定**：香典帳の設定

## レスポンシブ対応

### デスクトップ版

- **広い画面領域**を活用した効率的なレイアウト
- **サイドバー**による香典帳の素早い切り替え
- **キーボードショートカット**対応
- **ドラッグ&ドロップ**での直感的な操作

### モバイル版

- **タッチ操作**に最適化された画面設計
- **スワイプ操作**でのタブ切り替え
- **ボトムナビゲーション**による主要機能へのアクセス
- **モバイル専用メニュー**の表示

## キーボードショートカット

効率的な操作のためのショートカットキー：

### 全般
- `Ctrl + /`（Macは`Cmd + /`）：ショートカット一覧を表示
- `Esc`：ダイアログを閉じる
- `Tab`：次の要素にフォーカス移動

### 記録関連
- `Ctrl + N`（Macは`Cmd + N`）：新規記録を追加
- `Ctrl + S`（Macは`Cmd + S`）：保存
- `Ctrl + E`（Macは`Cmd + E`）：編集モード

### ナビゲーション
- `Ctrl + 1-5`（Macは`Cmd + 1-5`）：タブ切り替え
- `Ctrl + B`（Macは`Cmd + B`）：サイドバーの表示/非表示

## 検索とフィルタ機能

### 全体検索

1. **ヘッダーの検索ボックス**
   - 香典帳全体から横断検索
   - 名前、組織、備考欄などを対象

2. **検索オプション**
   - 完全一致 / 部分一致
   - 検索範囲の指定
   - 日付範囲での絞り込み

### 高度なフィルタ

1. **条件指定フィルタ**
   - 金額範囲
   - 関係性
   - 参列タイプ
   - 返礼状況

2. **並び替え機能**
   - 金額順（昇順/降順）
   - 日付順（最新/古い順）
   - 名前順（あいうえお順）

## データの表示形式

### 表示モードの切り替え

1. **リスト表示**
   - 情報を一覧で効率的に確認
   - 並び替えやフィルタに最適

2. **カード表示**
   - 詳細情報を見やすく表示
   - 視覚的に分かりやすい

3. **表形式**
   - スプレッドシートのような表示
   - データの比較や編集に便利

### 表示項目のカスタマイズ

- **列の表示/非表示**設定
- **列幅の調整**
- **表示順序**の変更
- **個人設定**として保存

## 操作のヒント

### 効率的な入力方法

1. **テンプレート機能**
   - よく使う情報をテンプレートとして保存
   - ワンクリックでの入力補完

2. **自動補完機能**
   - 過去の入力履歴から自動補完
   - 組織名や住所の効率的な入力

3. **一括操作機能**
   - 複数の記録を一度に編集
   - CSVファイルでの一括インポート

### 画面表示の最適化

1. **ズーム機能**
   - ブラウザのズーム機能対応
   - 文字サイズの調整

2. **ダークモード**
   - 目に優しい暗いテーマ
   - 設定から切り替え可能

## アクセシビリティ

### バリアフリー対応

- **スクリーンリーダー**対応
- **キーボードナビゲーション**完全対応
- **高コントラスト**モード
- **フォントサイズ**調整機能

### 多言語対応

- **日本語**メイン
- **英語**対応（一部機能）
- **言語設定**の保存

## トラブル時の対処

### よくある操作の問題

1. **ページが表示されない**
   - ブラウザの再読み込み
   - キャッシュのクリア

2. **保存できない**
   - 必須項目の確認
   - ネットワーク接続の確認

3. **動作が重い**
   - 不要なタブを閉じる
   - ブラウザの再起動
