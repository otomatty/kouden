# アカウント設定とプロフィール管理

## アカウント設定の概要

アカウント設定では、個人情報、セキュリティ、通知設定、表示設定など、アカウント全体に関わる重要な設定を管理できます。

## プロフィール情報の管理

### 基本情報の編集

1. **アクセス方法**
   - ヘッダー右上のユーザーアイコンをクリック
   - ドロップダウンメニューから「プロフィール」を選択

2. **編集可能な項目**
   - **表示名**：アプリ内で表示される名前
   - **メールアドレス**：ログインに使用するメールアドレス
   - **プロフィール画像**：アイコンとして使用される画像
   - **所属組織**：職場や組織名（任意）
   - **電話番号**：連絡先電話番号（任意）

3. **保存方法**
   - 必要な項目を編集後、「保存」ボタンをクリック
   - 変更内容は即座に反映されます

### プロフィール画像の設定

1. **画像のアップロード**
   - プロフィール画像エリアをクリック
   - ファイル選択ダイアログで画像を選択
   - 対応形式：JPEG、PNG、GIF（最大2MB）

2. **画像の編集**
   - アップロード後、トリミング機能で調整
   - プレビューで確認後、保存

## セキュリティ設定

### パスワードの変更

1. **変更手順**
   - セキュリティタブを選択
   - 「パスワード変更」セクションにアクセス
   - 現在のパスワードを入力
   - 新しいパスワードを入力（確認含む）
   - 「パスワードを変更」ボタンをクリック

2. **パスワードの要件**
   - 最低8文字以上
   - 英数字を含む
   - 特殊文字を含む（推奨）
   - 過去に使用したパスワードは避ける

### 二段階認証の設定

1. **有効化手順**
   - セキュリティタブの「二段階認証」セクション
   - 「有効にする」ボタンをクリック
   - QRコードを認証アプリでスキャン
   - 認証コードを入力して完了

2. **対応認証アプリ**
   - Google Authenticator
   - Microsoft Authenticator
   - Authy
   - その他TOTP対応アプリ

### ログインアクティビティの確認

- **最近のログイン履歴**を確認
- **不審なアクティビティ**の検出
- **セッション管理**：他のデバイスからのログアウト

## 通知設定

### 通知の種類

1. **香典帳関連**
   - 新しい記録の追加
   - 重複データの検出
   - メンバーからの変更

2. **メンバー関連**
   - 招待の受信
   - 権限の変更
   - 新しいメンバーの参加

3. **システム関連**
   - メンテナンス情報
   - 新機能のお知らせ
   - セキュリティアラート

### 通知方法の設定

1. **メール通知**
   - 通知タイプ別のオン/オフ設定
   - 即座に送信 / まとめて送信
   - 通知を受け取らない時間帯の設定

2. **アプリ内通知**
   - ブラウザプッシュ通知の許可
   - 通知音の設定
   - 表示時間の設定

## 表示設定

### テーマ設定

1. **カラーテーマ**
   - ライトモード（明るいテーマ）
   - ダークモード（暗いテーマ）
   - システム設定に従う（自動切り替え）

2. **フォント設定**
   - フォントサイズ（小・中・大）
   - フォントファミリー
   - 行間の調整

### 言語・地域設定

1. **表示言語**
   - 日本語
   - English（部分対応）

2. **地域設定**
   - 日付形式（和暦/西暦）
   - 通貨表示
   - 時刻形式（12時間/24時間）

### 操作設定

1. **キーボードショートカット**
   - ショートカットキーの有効/無効
   - カスタムショートカットの設定

2. **自動保存設定**
   - 自動保存の間隔
   - 下書き保存の設定

## プライバシー設定

### データの管理

1. **個人データの取り扱い**
   - データ使用目的の確認
   - 第三者提供の設定
   - 匿名化データの利用設定

2. **データのエクスポート**
   - 自分のデータをJSON形式でダウンロード
   - 香典帳データの一括エクスポート

3. **アカウントの削除**
   - アカウント削除の申請
   - データ削除の確認
   - 削除までの待機期間

### プライバシー設定の詳細

- **アクティビティの表示**：他のメンバーへの表示設定
- **オンライン状態**：オンライン状態の表示設定
- **検索設定**：検索結果への表示設定

## アカウント管理

### サブスクリプション管理

1. **プラン情報**
   - 現在のプラン確認
   - 使用量の確認
   - 請求履歴

2. **プラン変更**
   - アップグレード/ダウングレード
   - 支払い方法の変更
   - 自動更新の設定

### API設定

1. **API アクセス**
   - APIキーの生成・管理
   - アクセス権限の設定
   - 使用量の監視

## トラブルシューティング

### よくある問題

1. **メールアドレスが変更できない**
   - 現在のメールアドレスに確認メールが送信されます
   - 迷惑メールフォルダも確認してください

2. **パスワードを忘れた場合**
   - ログイン画面の「パスワードを忘れた方」をクリック
   - 登録メールアドレスにリセットリンクが送信されます

3. **二段階認証が使えない**
   - バックアップコードを使用してログイン
   - サポートに連絡して解除を依頼

### サポートへの連絡

アカウント関連で問題が発生した場合：

1. **サポートチケット**の作成
2. **ライブチャット**での即座の対応
3. **メールサポート**での詳細な問い合わせ

## セキュリティのベストプラクティス

### 安全なアカウント管理

1. **強固なパスワード**の使用
2. **二段階認証**の有効化
3. **定期的なパスワード変更**
4. **不審なアクティビティの監視**
5. **共有デバイスでの使用注意**
