# 電報の記録方法

## 電報記録について

電報記録は、葬儀や法要でいただいた弔電（電報）を管理する機能です。
差出人やメッセージ、香典・供物との関連を記録しておくことで、
後日のお礼やお返しの検討に役立ちます。

## 新しい電報を記録する

1. **電報タブにアクセス**
   - 香典帳の「電報」タブを開きます
   - 電報の一覧画面が表示されます

2. **登録フォームを開く**
   - 「弔電を登録する」ボタンをクリックします
   - 弔電の登録フォームが表示されます

3. **項目を入力して保存**
   - 必要な項目を入力し、「弔電を登録する」（または「追加」）で保存します
   - 保存すると「登録しました」と表示されます

## 入力項目

| 項目 | 必須 | 説明 |
| --- | --- | --- |
| **送信者名** | 必須 | 差出人のお名前（例：山田太郎、100文字以内） |
| **団体名** | 任意 | 会社名・団体名（例：株式会社〇〇） |
| **役職** | 任意 | 差出人の役職（例：代表取締役） |
| **メッセージ** | 任意 | 弔電の本文（1000文字以内） |
| **備考** | 任意 | 補足メモ（1000文字以内） |
| **関連する香典** | 任意 | 同じ差出人の香典記録と紐づけ |

## 関連する香典との紐づけ

差出人が香典もいただいている場合は、「関連する香典」で
該当する香典記録を選んで紐づけられます。
紐づけておくと、同じ方からの香典・電報・供物をまとめて把握できます。

## よくある質問

### Q: メッセージは必須ですか？
A: 必須ではありません。差出人名だけでも記録できます。後からメッセージや備考を追記することも可能です。

### Q: 1つの電報を複数の香典に紐づけられますか？
A: 電報には1件の香典記録を関連付けられます。

### Q: 記録した電報を編集・削除したい
A: [電報情報の管理](./telegram-management)をご覧ください。
