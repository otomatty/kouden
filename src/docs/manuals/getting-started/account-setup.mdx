# アカウント登録ガイド

香典帳アプリを利用するには、まずアカウントを作成する必要があります。
このページでは、**登録から初期設定まで** を詳しく説明します。

## 📝 アカウント作成の流れ

### ステップ1：サービスにアクセス

1. **ブラウザで香典帳アプリにアクセス**
   - パソコン：Chrome、Safari、Edge、Firefox
   - スマートフォン：Safari（iOS）、Chrome（Android）
   - タブレット：上記ブラウザの最新版

2. **「新規登録」ボタンをクリック**
   - トップページの右上にある青いボタン
   - または中央の「無料で始める」ボタン

### ステップ2：メールアドレスの登録

**入力項目**
- **メールアドレス**：ログイン時に使用するアドレス
- **パスワード**：8文字以上、英数字を含む強固なパスワード
- **パスワード確認**：上記パスワードの再入力

**重要な注意点**
- 📧 **有効なメールアドレスを使用**：確認メールが届きます
- 🔒 **安全なパスワードを設定**：他サービスと異なるパスワード推奨
- 📱 **普段使用するメールアドレス**：通知やサポート連絡用

### ステップ3：メール認証

1. **確認メールを受信**
   - 登録したメールアドレスに確認メールが送信されます
   - 通常1-2分以内に到着

2. **確認リンクをクリック**
   - メール内の「アカウントを有効化する」リンクをクリック
   - 自動的にログイン画面に移動

**メールが届かない場合**
- 📧 迷惑メールフォルダを確認
- ⏰ 5分程度待ってから再確認
- 🔄 「確認メール再送信」ボタンをクリック

### ステップ4：初回ログイン

1. **ログイン情報を入力**
   - メールアドレス
   - パスワード

2. **「ログイン」ボタンをクリック**

3. **ダッシュボードに移動**
   - 初回ログイン成功！

## ⚙️ 初期設定（プロフィール設定）

### 基本情報の設定

アカウント作成後、以下の基本情報を設定することをお勧めします：

**必須項目**
- **お名前**：香典帳の管理者名として表示
- **ふりがな**：正確な読み方

**任意項目**
- **電話番号**：緊急時の連絡先
- **住所**：お礼状の送り先など
- **所属・会社名**：組織での利用の場合

### プライバシー設定

**通知設定**
- 📧 **メール通知**：重要な更新をメールで受信
- 📱 **プッシュ通知**：リアルタイム通知（モバイル）
- 🔔 **システム通知**：アプリ内での通知

**共有設定**
- 🔒 **デフォルト権限**：新規作成時の共有レベル
- 👥 **チーム機能**：メンバー招待の許可設定

### セキュリティ設定

**パスワード管理**
- 🔐 **パスワード変更**：定期的な変更推奨
- 🛡️ **二要素認証**：セキュリティ強化（推奨）

**ログイン履歴**
- 📋 **アクセス記録**：不正ログインの確認
- 🌍 **接続地域**：海外からのアクセス監視

## 🔐 セキュリティの強化

### パスワードのベストプラクティス

**強いパスワードの条件**
- **8文字以上**：できれば12文字以上
- **大文字・小文字**：両方を含む
- **数字**：最低1つ以上
- **記号**：!@#$%^&*() など
- **辞書にない単語**：推測されにくい組み合わせ

**避けるべきパスワード**
- ❌ 生年月日や電話番号
- ❌ 辞書に載っている単語
- ❌ 他サービスと同じパスワード
- ❌ keyboard、password など簡単な単語

### 二要素認証の設定（推奨）

**設定手順**
1. **設定メニュー** → **セキュリティ**
2. **「二要素認証を有効にする」**をクリック
3. **認証アプリ**をスマートフォンにインストール
   - Google Authenticator
   - Microsoft Authenticator
   - Authy
4. **QRコードをスキャン**
5. **認証コードを入力**して設定完了

**メリット**
- 🛡️ **セキュリティ大幅向上**：パスワード流出時も安全
- 🔒 **不正アクセス防止**：第三者のログインを阻止
- 📱 **簡単な操作**：スマートフォンで簡単認証

## 📊 アカウントの種類

### 無料アカウント

**利用可能機能**
- 香典帳作成：**3冊まで**
- 記録数：**各香典帳100件まで**
- エクスポート：**基本形式のみ**
- メンバー招待：**2人まで**
- サポート：**平日のみ**

**制限事項**
- 高度な統計機能は利用不可
- 一括インポートは利用不可
- カスタマイズ機能は限定的

### 有料アカウント（プレミアム）

**追加機能**
- 香典帳作成：**無制限**
- 記録数：**無制限**
- エクスポート：**全形式対応**
- メンバー招待：**無制限**
- サポート：**24時間365日**

**高度な機能**
- 📊 **詳細統計・分析**
- 📁 **一括インポート・エクスポート**
- 🎨 **カスタマイズ機能**
- 🔄 **自動バックアップ強化**
- 👥 **チーム管理機能**

### 料金プラン

**無料プラン：￥0/月**
- 個人利用に最適
- 基本機能を無料で利用

**スタンダードプラン：￥980/月**
- 小規模なチームに最適
- 高度な機能を利用可能

**プレミアムプラン：￥1,980/月**
- 大規模な組織に最適
- 全機能を無制限で利用

## 🆙 アカウントのアップグレード

### アップグレードの手順

1. **設定メニュー** → **料金プラン**
2. **希望プランを選択**
3. **支払い情報を入力**
   - クレジットカード
   - PayPal
   - 銀行振込（年払いのみ）
4. **決済完了**後、即座に機能が有効化

### 支払い方法

**対応決済方法**
- 💳 **クレジットカード**：Visa、MasterCard、JCB、AMEX
- 💰 **PayPal**：安全な決済
- 🏦 **銀行振込**：年払いプランのみ対応

**請求サイクル**
- **月払い**：毎月自動更新
- **年払い**：2ヶ月分お得な料金
- **解約**：いつでもキャンセル可能

## ❓ よくある質問

### アカウント作成について

**Q: メールアドレスを間違えて登録してしまいました**
A: サポートにご連絡ください。確認後、正しいメールアドレスで再登録いたします。

**Q: パスワードを忘れてしまいました**
A: ログイン画面の「パスワードを忘れた方」をクリックし、パスワードリセットを行ってください。

**Q: 複数のメールアドレスでアカウントを作成できますか？**
A: 1つのメールアドレスに対して1つのアカウントのみ作成可能です。

### 料金・支払いについて

**Q: 無料期間はありますか？**
A: 無料アカウントは期間制限なしでご利用いただけます。有料プランは14日間の無料トライアルがあります。

**Q: 解約時に残り期間の返金はありますか？**
A: 月払いの場合、解約月の末日まで利用可能です。年払いの場合は、残り期間に応じて按分返金いたします。

**Q: 領収書は発行できますか？**
A: はい。設定画面から領収書のダウンロードが可能です。
