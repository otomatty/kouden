# 香典帳アプリとは

## ようこそ香典帳アプリへ

香典帳アプリは、葬儀や法要における香典・供物の管理を **デジタル化** し、
従来の手書きによる記録作業を大幅に効率化する専用ツールです。

大切な方を亡くされた際の複雑な手続きや記録作業を、
**正確で効率的** に行えるよう設計されています。

## なぜ香典帳アプリが必要なのか

### 従来の手書き管理の課題

従来の香典帳管理では、以下のような問題がありました：

- **手書きによる記録ミス**：金額や名前の書き間違い
- **文字が読めない**：急いで書いた文字が後で判読困難
- **集計作業の大変さ**：手計算による合計金額の算出
- **お礼状作成の手間**：住所の手入力や宛名書き
- **データの紛失リスク**：紙の紛失や汚損
- **複数人での管理困難**：同時に複数人が記録できない

### デジタル化のメリット

香典帳アプリを使用することで、これらの課題が解決されます：

1. **正確性の向上**
   - 入力時の自動チェック機能
   - 重複データの自動検出
   - 計算ミスの防止

2. **効率性の大幅向上**
   - 入力の高速化
   - 自動集計・統計
   - 一括処理機能

3. **データの安全性**
   - クラウドでの自動バックアップ
   - データの暗号化保存
   - アクセス権限の管理

4. **チームでの協力作業**
   - 複数人での同時入力
   - リアルタイム同期
   - 役割分担機能

## 香典帳アプリの主要機能

### 1. 香典・供物の記録管理

- **香典記録**：金額、送り主、関係性、住所などの詳細管理
- **供物記録**：供花、弔電、果物等の記録と写真管理
- **参列情報**：参列方法（葬儀参列、弔問、欠席）の記録

### 2. 効率的なデータ入力

- **重複チェック**：同一人物からの重複を自動検出
- **一括インポート**：CSVファイルからの大量データ取り込み
- **自動補完**：過去の記録からの自動入力候補

### 3. 高度な検索・分析機能

- **柔軟な検索**：名前、金額、関係性など様々な条件での検索
- **統計分析**：関係性別、金額帯別などの詳細分析
- **グラフ表示**：視覚的なデータ分析

### 4. レポート・出力機能

- **PDF出力**：印刷可能な美しいレポート生成
- **Excel/CSV出力**：他システムとの連携
- **お礼状作成支援**：宛名ラベルの自動生成

### 5. チーム機能

- **メンバー招待**：家族や関係者の協力作業
- **権限管理**：閲覧のみ、編集可能などの細かい権限設定
- **履歴管理**：誰がいつ何を変更したかの記録

## セキュリティとプライバシー

### データ保護への取り組み

香典帳には **機密性の高い個人情報** が含まれるため、
厳重なセキュリティ対策を実施しています：

- **暗号化通信**：すべてのデータ通信をSSL/TLSで暗号化
- **データ暗号化**：保存されるデータは暗号化して保管
- **アクセス制御**：多要素認証とアクセス権限管理
- **定期バックアップ**：データ消失防止の自動バックアップ
- **プライバシー保護**：個人情報保護法に完全準拠

## よくある最初の質問

### Q: パソコンが苦手でも使えますか？
A: はい。直感的な操作で、スマートフォンを使える方なら簡単にご利用いただけます。

### Q: データが消える心配はありませんか？
A: クラウドで自動保存・バックアップされるため、安心してご利用ください。

### Q: 他の人に見られる心配はありませんか？
A: 招待した方以外はアクセスできません。プライバシーは完全に保護されます。

### Q: 費用はどのくらいかかりますか？
A: 基本機能は無料でご利用いただけます。詳細は料金プランをご確認ください。
