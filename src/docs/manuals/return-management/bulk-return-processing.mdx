# 一括返礼処理

## 一括返礼処理とは

一括返礼処理は、同じ香典金額のご芳名をグループにまとめて、
返礼品や返礼状況を一度に設定できる機能です。
「3,000円の方にはこの返礼品」「5,000円の方にはこの返礼品」というように、
金額帯ごとの返礼方針が決まっている場合に大幅に作業を短縮できます。

## 一括変更を開く

1. 香典帳の「返礼」タブを開きます
2. 「一括変更」ボタンをクリックします
3. 「一括更新（テーブル形式）」のダイアログが表示されます
   - 金額ごとにグループ化された返礼記録が一覧で表示されます

## 画面に表示される情報

ダイアログ上部には、対象データの概要が表示されます。

- **総件数**：対象となる返礼記録の件数
- **金額種類**：いくつの金額帯に分かれているか
- **総金額**：対象全体の合計金額

テーブルには金額帯ごとに次の列が並びます。

- **金額**：その金額帯
- **件数**：その金額帯に該当するご芳名の数
- **返礼品のチェックボックス**：割り当てる返礼品を選択
- **選択コスト**：選んだ返礼品の費用
- **損益**：いただいた金額に対する損益
- **ステータス**：返礼状況（未対応／一部返礼／完了／返礼不要）

## 一括で設定する

1. **金額帯を選択**
   - 設定したい金額グループを選びます

2. **返礼品を割り当て**
   - その金額帯に渡す返礼品をチェックボックスで選択します
   - すべての項目に対して一括で選択することもできます

3. **返礼状況を設定**
   - 必要に応じて返礼状況をまとめて変更します

4. **更新を実行**
   - 設定内容を確認して一括更新を実行します
   - 完了すると、更新された件数とともに「一括更新完了」が表示されます

## よくある質問

### Q: 金額が同じでも個別に返礼品を変えたい場合は？
A: 一括処理で大まかに設定したあと、[返礼記録の管理](./return-records)で個別に上書き編集できます。

### Q: 一括変更を取り消せますか？
A: 一括更新の取り消し（アンドゥ）機能はありません。設定内容をよく確認してから実行してください。誤って変更した場合は、再度正しい内容で更新し直してください。

### Q: どの金額帯にどの返礼品を割り当てるか整理したい
A: [返礼品マスター](./return-items-setup)で推奨香典金額を設定しておくと、金額帯と返礼品の対応を整理しやすくなります。
