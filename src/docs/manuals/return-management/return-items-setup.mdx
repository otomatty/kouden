# 返礼品マスターの設定

## 返礼品マスターとは

返礼品マスターは、返礼でお渡しする品物（カタログギフト、会葬品、食品など）を
あらかじめ登録しておく機能です。
ここに登録した返礼品は、後述の[返礼記録の管理](./return-records)で選択して
各ご芳名に割り当てられます。よく使う品物を登録しておくことで、
返礼作業を効率化できます。

## 返礼品マスターを開く

1. **返礼品管理画面にアクセス**
   - 香典帳の「返礼」タブを開きます
   - 「返礼品を管理する」ボタンをクリックすると、返礼品マスターの一覧が表示されます

2. **表示形式の切り替え**
   - 一覧はグリッド表示・リスト表示を切り替えられます
   - カテゴリでの絞り込み、有効・無効での絞り込み、キーワード検索に対応しています

## 返礼品を登録する

1. **新規登録**
   - 「返礼品を追加」から登録フォームを開きます

2. **項目を入力**
   - **返礼品名**（必須）：品物の名称（100文字以内）
   - **説明**：品物の内容や特徴（500文字以内）
   - **価格**：1点あたりの金額（0円〜10,000,000円）
   - **カテゴリ**：以下から選択します
     - **会葬品**
     - **ギフト券**
     - **食品**
     - **花・植物**
     - **その他**
   - **画像**：返礼品の写真（任意）
   - **推奨香典金額（最小・最大）**：この返礼品をおすすめする香典金額の目安（任意）
   - **表示順序**：一覧での並び順（1〜9999）

3. **有効・無効の設定**
   - 「有効」にした返礼品が返礼記録での選択候補に表示されます
   - 一時的に使わない返礼品は「無効」にしておくと一覧から隠せます

4. **保存**
   - 入力内容を確認して保存します

## 返礼品の編集・削除

- 一覧から各返礼品を選び、内容の編集や削除ができます
- 有効・無効の切り替えもこの画面から行えます

## 推奨香典金額の活用

推奨香典金額（最小・最大）を設定しておくと、
「この金額帯にはこの返礼品」という目安を整理できます。
香典金額に応じた返礼品選びの基準として活用してください。

## よくある質問

### Q: 返礼品はいくつでも登録できますか？
A: 件数の上限はありません。よくお渡しする品物をまとめて登録しておくと便利です。

### Q: 一度登録した返礼品を使わなくなった場合は？
A: 削除するか、「無効」に切り替えてください。無効にすると返礼記録の選択候補から外れますが、データは残ります。

### Q: 価格は必須ですか？
A: 必須ではありませんが、価格を登録しておくと返礼記録で返礼品費用や損益が自動計算されます。
