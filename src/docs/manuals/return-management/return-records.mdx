# 返礼記録の管理

## 返礼記録について

返礼記録は、香典をいただいた方それぞれに対する返礼の状況を管理する機能です。
香典記録（エントリー）と連動しており、ご芳名・団体名・関係性・香典金額などを
確認しながら、返礼状況や返礼品を記録できます。

## 返礼記録を開く

1. 香典帳の「返礼」タブを開きます
2. 一覧には登録済みのご芳名が並び、各行に返礼状況などが表示されます
   - パソコンではテーブル形式、スマートフォンではカード形式で表示されます

## 返礼状況の種類

返礼記録では、以下の4つの状況で進捗を管理します。

| 状況 | 意味 |
| --- | --- |
| **未対応** | まだ返礼の手配をしていない状態 |
| **一部返礼** | 一部の返礼が完了している状態 |
| **完了** | 返礼がすべて完了した状態 |
| **返礼不要** | 返礼の必要がない（辞退された場合など） |

一覧の返礼状況は、その場で切り替えることができます。

## 返礼情報を編集する

各ご芳名の行から「返礼情報を編集する」を開くと、
以下のタブに分かれた入力画面が表示されます。

### 基本情報タブ
- **返礼状況**：未対応／一部返礼／完了／返礼不要から選択
- **葬儀ギフト金額**：当日返しなどの金額

### 返礼品タブ
- [返礼品マスター](./return-items-setup)に登録した品物を選択して割り当てます
- 選択した返礼品に応じて返礼品費用や損益が計算されます

### 詳細タブ
- **手配日**：返礼を手配した日付
- **備考**：補足メモ
- **発送先情報**：郵便番号・住所・電話番号（[配送状況の管理](./delivery-tracking)を参照）

## 一覧で確認できる情報

返礼記録の一覧では、次のような情報をまとめて確認できます。

- 登録日時、名前、組織、役職、関係性
- 香典金額、お供物配分、合計金額
- 返礼状況、葬儀ギフト金額、追加返礼金額
- 返礼方法、返礼品、返礼品費用、損益
- 発送先、手配日、備考

## 配分管理

香典に加えて供物をいただいている場合、
「配分管理」から供物分の金額を各ご芳名に配分し、
合計金額をもとにした返礼の検討ができます。

## よくある質問

### Q: 返礼記録は手動で新規作成しますか？
A: 返礼記録は香典記録（ご芳名）と連動しています。香典を登録すると返礼記録の対象として一覧に並び、そこに返礼情報を追記していく形になります。

### Q: 一度に多くの返礼状況を変更したい
A: [一括返礼処理](./bulk-return-processing)で、金額ごとにまとめて返礼品や状況を設定できます。

### Q: 返礼品費用や損益はどう計算されますか？
A: 返礼品マスターに登録した価格と、いただいた金額をもとに自動計算されます。
