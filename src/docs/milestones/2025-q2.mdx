# 2025年第2四半期の開発計画

第1四半期で実現した快適性と安定性を基盤に、この四半期では**モバイル体験の飛躍的向上**を目標とします。現在のPWA実装をさらに発展させ、ネイティブアプリの提供、AI機能による作業効率化、より充実したオフライン対応により、「いつでも、どこでも、快適に」香典管理ができる環境を実現します。

忙しい時期だからこそ、スマートフォンでの操作性を最優先に考え、外出先でもストレスなく使える香典帳アプリを目指します。

<Accordion title="AIによる香典袋自動読み取り機能">

### 撮影するだけで自動入力
香典袋を撮影するだけで、以下の情報を自動で読み取り、入力フォームに反映します。手書き文字にも対応し、入力時間を大幅に短縮できます。

**自動読み取り対象**
- **ご芳名**：送り主のお名前
- **金額**：包まれた金額（袋に記載がある場合）
- **住所**：記載されている住所情報
- **会社名・肩書き**：組織名や役職

### 高精度なAI認識エンジン
**文字認識精度**：95%以上の高精度を実現
**対応書体**：楷書、行書、草書体に対応
**複数言語**：日本語、英語、中国語に対応
**リアルタイム処理**：撮影後3秒以内に結果表示

### スマート補正機能
認識結果に曖昧な部分がある場合、過去のデータベースから類似パターンを検索し、最適な候補を提案します。また、明らかに間違いがある場合は、修正候補を自動表示します。

</Accordion>
